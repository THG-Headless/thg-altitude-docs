# Invisible CAPTCHA Examples

Invisible CAPTCHAs provide seamless bot protection without disrupting the user experience. They run in the background and only show a challenge when suspicious activity is detected.

Unlike visible CAPTCHAs, invisible ones require manual execution using the `execute()` method, typically triggered on form submission. **Note:** The `onExpired` callback is typically not needed for invisible CAPTCHAs since `execute()` generates fresh tokens each time.

## Configuration

Set `size: invisible` in your Captcha config for reCAPTCHA and HCaptcha to enable invisible mode:

```typescript
const captcha = new Captcha({
  provider: "recaptcha", // or 'hcaptcha'/'turnstile'/'friendlyCaptcha'
  siteKey: "your-site-key",
});
```

**Note:** Turnstile doesn't require the `invisible` flag - its behavior is configured in the Cloudflare dashboard. Friendly Captcha works similarly to invisible mode by default, using the `startMode` option to control when puzzles are solved.

```typescript
import { Captcha } from "@thg-altitude/captcha";

const invisibleCaptcha = new Captcha({
  provider: "recaptcha",
  siteKey: "your-invisible-recaptcha-key",
});

await invisibleCaptcha.render("container-id", "form-protection", {
  size: "invisible",
});

// Execute on form submission
document.getElementById("submit-btn").onclick = async () => {
  try {
    await invisibleCaptcha.execute("form-protection");
    const token = await invisibleCaptcha.getToken("form-protection");

    // ✅ Recommended - automatic submission on token
    await submitForm(token);
  } catch (error) {
    console.error("CAPTCHA verification failed:", error);
    // Handle error (show message, etc.)
  }
};
```

## Framework Integration Examples
For framework integrations, it's recommended to pass your form submission function to the `onToken` callback for cleaner code flow and prevent any race conditions that may occur by trying to use the token before the callback is complete.

### React

```typescript
import { useEffect, useRef } from "react";
import { Captcha } from "@thg-altitude/captcha";

function ContactForm() {
  let captcha;
  const captchaId = "contact-form";
  const containerId = "contact-form-recaptcha";

  useEffect(() => {
    captcha = new Captcha({
      provider: "recaptcha",
      siteKey: "your-site-key",
    });

    await captcha.render(containerId, captchaId, {
      size: type,
      onToken: (captchaToken) => {
        submitForm(captchaToken)
      },
      onError: () => {
        console.error("Error rendering captcha:", captchaId);
      },
      onExpired: () => {
        setToken(null);
      },
    });
  }, []);

  const submitForm = async (token) => {
    try {
      // ✅ Recommended - automatic submission on token
      await fetch(..., token)
    } catch (error) {
      console.error("Fetch error", error);
    }
  };

  const handleSubmit = async (e: React.FormEvent) => {
    if (!captcha) return;

    try {
      // the callback passed to execute submits the form when the token is available
      await captchaRef.current.execute("contact-form");
      console.log("CAPTCHA token:", token);
    } catch (error) {
      console.error("CAPTCHA failed:", error);
    }
  };

  return (
    <form onSubmit={handleSubmit}>
      <input type="text" placeholder="Name" required />
      <input type="email" placeholder="Email" required />
      <textarea placeholder="Message" required></textarea>

      <div id={containerId}></div>

      <button type="submit">Submit</button>
    </form>
  );
}
```

## Friendly Captcha with Manual Execution

Friendly Captcha can work in an invisible-like mode using `startMode: 'none'` which requires manual execution. The widget is created using the Friendly Captcha SDK and listens for widget events:

```typescript
import { Captcha } from "@thg-altitude/captcha";

const friendlyCaptcha = new Captcha({
  provider: "friendlyCaptcha",
  siteKey: "your-friendly-captcha-site-key",
});

// Render with manual start mode
await friendlyCaptcha.render("captcha-container", "form-protection", {
  startMode: "none", // Requires manual execution via execute()
  onToken: (token) => {
    // Called when frc:widget.complete event fires
    // Token received from event.detail.response
    submitForm(token);
  },
  onError: () => {
    // Called when frc:widget.error event fires
    console.error("Friendly Captcha verification failed");
  },
  onExpired: () => {
    // Called when frc:widget.expire event fires
    console.log("Friendly Captcha token expired");
  }
});

// Execute on form submission - calls widget.start()
document.getElementById("submit-btn").onclick = async () => {
  try {
    await friendlyCaptcha.execute("form-protection");
    // Token will be provided via onToken callback when puzzle completes
  } catch (error) {
    console.error("CAPTCHA verification failed:", error);
  }
};
```

### React Example with Friendly Captcha

```typescript
import { useEffect } from "react";
import { Captcha } from "@thg-altitude/captcha";

function ContactForm() {
  let friendlyCaptcha;
  const captchaId = "contact-form";
  const containerId = "friendly-captcha-container";

  useEffect(() => {
    friendlyCaptcha = new Captcha({
      provider: "friendlyCaptcha",
      siteKey: "your-friendly-captcha-site-key",
    });

    friendlyCaptcha.render(containerId, captchaId, {
      startMode: "none", // Manual execution - widget won't start until execute() is called
      onToken: (captchaToken) => {
        // Event handler for frc:widget.complete
        submitForm(captchaToken);
      },
      onError: () => {
        // Event handler for frc:widget.error
        console.error("Friendly Captcha error:", captchaId);
      },
      onExpired: () => {
        // Event handler for frc:widget.expire
        console.log("Friendly Captcha token expired:", captchaId);
      }
    });
  }, []);

  const submitForm = async (token) => {
    try {
      await fetch('/api/contact', {
        method: 'POST',
        headers: {
          'Content-Type': 'application/json',
          'X-Captcha-Response': token
        },
        body: JSON.stringify(formData)
      });
    } catch (error) {
      console.error("Form submission failed:", error);
    }
  };

  const handleSubmit = async (e: React.FormEvent) => {
    e.preventDefault();
    if (!friendlyCaptcha) return;

    try {
      await friendlyCaptcha.execute(captchaId);
      // Form submission happens in onToken callback
    } catch (error) {
      console.error("Friendly Captcha failed:", error);
    }
  };

  return (
    <form onSubmit={handleSubmit}>
      <input type="text" placeholder="Name" required />
      <input type="email" placeholder="Email" required />
      <textarea placeholder="Message" required></textarea>

      <div id={containerId}></div>

      <button type="submit">Submit</button>
    </form>
  );
}
```

## Turnstile specific
By default Turnstile will execute on render, to defer this execution to mimic an invisible-stlye mechanism seen in hCaptcha and reCAPTCHA you can specify the execution mode to be `execution: 'execute'`.
This means execute must be called manually.

```typescript
import { Captcha } from "@thg-altitude/captcha";

const ContactForm = () => {
  let turnstile;
  const [form, setForm] = useState({ name: '', email: '', message: '' });

  useEffect(() => {
    // Initialize invisible Turnstile widget
    turnstile = new Captcha('turnstile', 'your-site-key');
    turnstileRef.current.render('turnstile-container', 'form-captcha', {
      appearance: 'execute',
      execution: 'execute',
      onToken: (token) => {
        // Token received, now submit the form
        submitForm(token);
      }
    });
  }, []);

  const handleSubmit = async (e) => {    
    // Trigger CAPTCHA verification before form submission
    await turnstileRef.current.execute('form-captcha');
  };

  const submitForm = async (captchaToken) => {
    // Submit form with CAPTCHA token
    const response = await fetch('/api/contact', {
      method: 'POST',
      body: JSON.stringify({ ...form, captchaToken })
    });
  };

  return (
    <form onSubmit={handleSubmit}>
      <input type="text" placeholder="Name" required />
      <input type="email" placeholder="Email" required />
      <textarea placeholder="Message" required></textarea>
      
      <div ref={containerRef} id="captchaContainer" />
      
      <button type="submit" >
        Submit
      </button>
    </form>
  );
};
```
