# Google Analytics and Custom Event Schemas 

## Checkout Schema

### Apply Coupon Success Schema
**Event Name:** `apply_coupon_success`  
**Parameters:**
```typescript
{
  coupon_code: string,                   // Required - The coupon code that was successfully applied
  eventData: {                           // Optional - Additional event tracking data
    eventCategory: string,               // Category of the event ("Discount codes")
    eventAction: string,                 // Action taken (e.g. "Valid")
    eventLabel: string,                  // Label for the event (typically the coupon code)
    eventPage: string                    // Page where the coupon was applied
  }
}
```

### Apply Coupon Fail Schema
**Event Name:** `apply_coupon_fail`  
**Parameters:**
```typescript
{
  coupon_code: string,                   // Required - The coupon code that failed to apply
  validation_errors: string,             // Required - Type of validation error that occurred
  eventData: {                           // Optional - Additional event tracking data
    eventCategory: string,               // Category of the event ("Discount codes")
    eventAction: string,                 // Action taken (e.g., "Inactive", "Invalid")
    eventLabel: string,                  // Label for the event (typically the coupon code)
    eventPage: string                    // Current page path where the coupon was applied
  }
}
```

## Ecommerce Schema

### Base ecommerce schemas
Many of the ecommerce schemas share the same base schemas including:
### Base product item schema
```typescript
{
  item_id: string,                     // Required - Unique identifier for the item
  item_name: string,                   // Required - Name of the item
  item_brand: string,                  // Optional - Brand of the item
  price: number,                       // Optional - Price of the item
  quantity: number,                    // Optional - Quantity of the item
  index: number,                       // Optional - Position of the item in a list
  discount: number,                    // Optional - Discount amount applied to item
  affiliation: string,                 // Optional - Store or affiliation
  coupon: string,                      // Optional - Coupon code applied to item
  item_category: string,               // Optional - Category of the item
  item_list_name: string,              // Optional - Name of list where item appears
  item_variant: string                 // Optional - Variant title of the item
}
```

### Base ecommerce schema
```typescript
{
  currency: string,                      // Required - Currency code (e.g., "GBP", "USD")
  value: number,                         // Required - Sum of (price * quantity)
  items: Array<BaseProductItem>          // Required - Array of products in the transaction
}
```

### Basket Item Schema
```typescript
{
  sku: string,                          // Required - Product SKU identifier
  quantity: number,                     // Required - Number of this item
  price: number,                        // Required - Regular price of the item
  vipPrice: number,                     // Optional - VIP/Member price if applicable
  isFreeGift: boolean                   // Optional - Whether item is a free gift
}
```

## Miscellaneous Events

### Page Not Found Schema
**Event Name:** `page_not_found`  
**Parameters:**
```typescript
{
  eventData: {                           // Required - Event tracking data
    eventCategory: string,               // Optional - Fixed value "Errors"

  }
}
```

**Event Name:** `logout`  
**Parameters:** None

**Event Name:** `sign_up`  
**Parameters:**
```typescript
{
```

**Event Name:** `user_details`  
**Parameters:**
```typescript
{
  visitorLoginState: 'LOGGED_IN' | 'LOGGED_OUT',  // Required - Current login state

}
```

**Parameters:**
```typescript
{
  currency: string,                      // Required - Currency code (e.g., "GBP", "USD")

  product_id: string,                    // Optional - Product identifier
  product_name: string,                  // Optional - Name of the product

    eventAction: string,                 // Required - Action being performed
    eventLabel: string,                  // Required - Item identifier
    eventPage: string                    // Required - Current page path
  }
}
```

```typescript
{
  currency: string,                      // Required - Currency code (e.g., "GBP", "USD")
  value: number,                         // Required - Total value of items
```
