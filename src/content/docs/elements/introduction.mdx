### What are Elements

Altitude's suite of modular UI of ready made commerce functionality with pre-optimised user experience and accessibility. We built Elements to pair with Ingenuity's Commerce APIs to quickly get started with complex components without the headaches.

Elements features include:

- • Flexible Configuration and UI.
- • Performance, user experience and accessibility optimised.
- • Framework-less Reactivity and Framework Variants.
