# Astro Integration

The Astro Integration provides a comprehensive toolkit for building high-performance e-commerce websites. It seamlessly integrates with Astro sites to unlock powerful commerce capabilities, multi-tenant architecture, and global i18n support—all optimised for edge performance.

- 👯‍♂️ **[Multitenancy](./v3.0.0/reference/multi-tenancy)** Support for multi-tenancy via GTLD-based tenant-switching.
- 🌎 **[i18n](./v3.0.0/reference/i18n):** Support for i18n via [GTLD](https://en.wikipedia.org/wiki/Generic_top-level_domain)-based localisation, powered by the Altitude Platform [Edge KV store](http://localhost:8080/docs/platform/v2.3.0/edge/kv-store)
- 🚀 **Performance Patterns:** Edge performance optimisations out of the box.
- 🔌 **[Commerce APIs](./v3.0.0/reference/commerce-api):** Access to the commerce API with ready-made helpers and proxies
