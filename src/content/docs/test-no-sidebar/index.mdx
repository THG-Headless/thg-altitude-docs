# Test Page Without Sidebar

This page demonstrates how the layout looks when there's no sidebar content. The sidebar should be hidden, and the main content should be centered with appropriate padding.

## Main Features

- Sidebar is automatically hidden when there's no content
- Main content is centered with appropriate padding
- Layout structure remains consistent with other pages
- No manual configuration needed

## How It Works

The solution works by:

1. The DocsSidebar component checks if there's any content to display
2. If there's no content, it renders nothing
3. The DocsLayout component checks if the sidebar slot has content
4. If the sidebar slot is empty, it adjusts the layout accordingly

## Benefits

This approach has several benefits:

1. **Automatic Detection**: No need to manually specify whether to show the sidebar
2. **Consistent Layout**: The overall layout structure remains the same
3. **Improved User Experience**: Pages without navigation don't have empty sidebars
4. **Maintainable Code**: The solution is implemented in a clean, maintainable way
