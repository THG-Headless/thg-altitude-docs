# Insights v0.6

A lightweight analytics library designed for modern web applications. Insights provides automatic tracking with minimal setup - just add the component and set your tenant to start collecting comprehensive analytics data.

The insights package forms the client side SDK for the new THG Ingenuity tracking pipeline, providing data to drive reporting.

The insights package provides out the box tracking of page view events. Since the tracking data is primarily driven by back end events, this should be sufficient for most use cases. For cases where a front event needs tracking that does not have a back end event, the package provides HTML attributes and a JavaScript API as means to trigger tracking events.

## Key Features

- **Minimal Setup**: 1 line setup to start tracking
- **Automatic Page Views**: Tracks navigation and core user metrics with zero configuration
- **HTML Attribute Tracking**: Track clicks and impressions automatically with `data-insights-*` attributes
- **Framework Agnostic**: Works with any JS front end web application framework.
- **Zero Configuration**: Authentication, batching, and transmission handled automatically
- **Debug Mode**: Development builds with detailed logging and diagnostics
- **Lightweight**: ~30KB lightweight production bundle minimises performance impact
