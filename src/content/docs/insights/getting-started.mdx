## Installation

Install the package using your preferred package manager:

```bash
# Using npm
npm install @thg-commerce/insights

# Using yarn
yarn add @thg-commerce/insights

# Using pnpm
pnpm add @thg-commerce/insights
```

## Configuration

Configure Insights with your endpoint and options:

```typescript
import { createTracker } from '@thg-commerce/insights';

const tracker = createTracker({
  endpoint: 'https://api.example.com/events',
  domain: 'my-app',
  debug: true // Enable for development
});
```

### Framework-Specific Setup

```typescript
// Astro
import insights from '@thg-commerce/insights/astro';

// Next.js
import insights from '@thg-commerce/insights/next';

// React Native
import insights from '@thg-commerce/insights/react-native';

// Initialize with framework-specific integration
insights({
  endpoint: 'https://api.example.com/events',
  debug: true
});
```
