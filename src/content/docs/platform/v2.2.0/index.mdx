# Altitude Platform v2.2.0

Welcome to the documentation for Altitude Platform version 2.2.0. This version includes performance improvements and bug fixes compared to v2.1.0.

## What's New in v2.2.0

- **Performance Optimizations**: Significant performance improvements for edge functions
- **Enhanced KV Store**: Improved KV store with better caching and performance
- **Bug Fixes**: Fixed several issues reported in v2.1.0
- **Improved Logging**: Enhanced logging capabilities with better filtering
- **UI Improvements**: Redesigned dashboard with better usability
